Trim

Trim without losing the thread.

safe (the default) keeps your last 5 exchanges fully intact: tool outputs, thinking, file contents. Older turns get observation-masked — tool call names and file paths stay as breadcrumbs, bodies drop. Research-aligned with JetBrains' 2025 observation-masking study.

Need more? smart applies per-component rules by turn depth (~45% saved). slim drops older breadcrumbs for ~70% saved. archive is historical only.

Use ccw, the claudecompress wrapper, for a seamless resume. Or run bunx claudecompress as a standalone script.

Savings

Real numbers, real session.

Measured on a 761k-token Opus 4.6 session (153 user turns). Cost is the real cold /resume rebuild — 1h cache write rate ($10/Mtok on Opus 4.6, per Anthropic pricing). safe (default) keeps the last 5 exchanges fully intact and observation-masks everything older — the pattern validated by JetBrains' 2025 NeurIPS paper. All modes apply squash (rtk-style per-tool-call compression) to preserved tool outputs. Run bunx claudecompress on your own sessions to verify the math.
